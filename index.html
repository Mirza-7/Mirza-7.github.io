Featured Projects

Coin Sorter Simulator

This project is a C program designed to simulate a Coin Sorting Machine. The program sorts up to five different types of coins based on their weight, diameter, condition (bent or usable), and image recognition. It allows users to configure the sorter for any group of coins, such as Canadian coins, by inputting a configuration file that defines each coin type’s attributes.

Music Playlist Tracker

Playlist Tracker that allows users to add, remove, swap, and manage songs dynamically. The program uses linked lists to store the playlist, ensuring efficient insertion, deletion, and modifications. Users can interactively modify their playlist by choosing from various commands to customize the song order and structure.

Unbeatable Tic Tac Toe AI

This Python program is a Tic-Tac-Toe game where the AI is programmed to always win or at least force a draw. The AI uses the Minimax algorithm, a decision making algorithm commonly used in games, to ensure optimal moves every time. The game allows a human player to compete against the AI, but beating it is impossible!
